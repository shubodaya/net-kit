    Packet capture prerequisites
    • Install Npcap (driver) from nmap.org, or use the bundled installer when prompted.
    • Run Net Kit as Administrator to allow driver access.
    • If TShark is present, it is used as fallback; otherwise capture uses Npcap directly.
    • If no interfaces appear, click “Refresh interfaces”, ensure Npcap is installed, and restart the app.
